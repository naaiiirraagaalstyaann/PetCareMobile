# PawCare 🐾

## The Problem
In my house, we often forget if the dog has been fed or if someone already took them 
for a walk. This leads to double-feedings or missed tasks.

## The Solution
I built this mobile app using **C#** and **.NET MAUI** to act as a simple log. It lets 
anyone in the family check off tasks so the rest of us can see exactly what has been 
done for our pet today.

## Key Features
- **Quick Log:** Enter your name, pet name, and select an activity (Feeding, Medication, Walk, Grooming)
- **History:** A timestamped list of all activities with who completed them
- **Filter:** "Show today only" checkbox to focus on today's tasks
- **Auto-Save:** Uses JSON to save your data directly on your phone

## Technologies Used
- C# / .NET MAUI
- Android Emulator (tested on API level 5554)
- JSON for local data persistence
- Git / GitHub for version control

## How to Run
1. Clone the repository
2. Open `PetCareMobile.sln` in Visual Studio
3. Set the target to **Android Emulator**
4. Press **Run**
